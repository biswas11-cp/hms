# 🚀 Hospital Management System – Spring Boot

## 📌 Overview
A Full Stack web application developed using Spring Boot that provides RESTful APIs for managing Hospital data.

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Oracle SQL
- Maven
- Postman (API Testing)

## 🔥 Features
- Add Doctors,Patients,Appointments
- Update Doctors,Patients,Appointments
- Delete Doctors,Patients,Appointments
- Get All Doctors,Patients,Appointments
- Exception Handling
- Layered Architecture (Controller, Service, Repository)

## 📂 Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- Model (Entity)

## ⚙️ How to Run
1. Clone the repository
2. Configure Oracle SQL in application.properties
3. Run as Spring Boot Application

## 📸 API Testing
Tested using Postman
